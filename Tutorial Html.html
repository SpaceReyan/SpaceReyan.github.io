HTML for Beginners

A simple and interactive guide to get you started with HyperText Markup Language.

1. What is HTML?

HTML stands for HyperText Markup Language. It's the standard language for creating web pages. Think of it as the **skeleton of a webpage**â€”it provides the structure and content.

HTML uses "tags" to mark up different parts of the content, like headings, paragraphs, and lists. These tags are enclosed in angle brackets, for example, <p> for a paragraph.

2. Basic Document Structure

Every HTML page follows a basic structure. Copy and paste the code below into an editor to see it in action. 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first paragraph.</p>
</body>
</html>

This code defines the core parts of a page: the `html` tag for the whole document, the `head` for metadata, and the `body` where all the visible content goes.

3. Common HTML Tags

Let's explore some of the most frequently used tags.

Headings: <h1> to <h6>

Used for titles and subtitles. `<h1>` is the most important.

This is an h1 heading

This is an h2 heading

Paragraphs: <p>

Used for blocks of text.

This is a simple paragraph of text. It's great for writing content.

Links: <a>

Used to create hyperlinks to other pages. The `href` attribute specifies the destination.

Click me to go to Google!

Images: <img>

Used to embed images. The `src` attribute is the image's source URL, and `alt` provides alternative text.
